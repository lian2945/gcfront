# GC Board Next.js Client

Next.js 기반의 실제 애플리케이션 같은 클라이언트입니다. 이메일 인증/회원가입/로그인에서부터 게시글 작성·검색, 댓글 작성까지 하나의 화면에서 이어집니다.

## 빠른 실행
```bash
cd frontend
npm install
npm run dev
```
뒤에 나오는 브라우저 주소(보통 `http://localhost:3000`)에서 열면 됩니다. API Base URL을 상단 세션 카드에서 `http://localhost:8080` 등으로 맞춰주세요.

## 주요 흐름
1. 이메일 인증/회원가입을 순서대로 진행하고, 로그인하면 JWT를 로컬스토리지에 저장합니다.
2. 피드에서 글을 선택하면 상세가 오른쪽에 표시되고, 좋아요/수정/삭제·댓글 작성 기능을 제공합니다.
3. 댓글도 커서 기반으로 더 보기 버튼을 눌러 읽을 수 있으며, 대댓글은 부모 ID 입력으로 작성합니다.
4. API Base를 변경하면 다른 백엔드를 가리킬 수 있고, 로그아웃 시 토큰이 제거됩니다.

## 정적 파일
이 앱은 Next.js를 통해 별도 빌드/서버로 운영됩니다. `npm run build` 후 `npm run start`로 배포할 수 있으며, `frontend/.next` 폴더를 Spring 정적 자원에 복사하거나 리버스 프록시로 연결하면 전체 서비스로 사용할 수 있습니다.
